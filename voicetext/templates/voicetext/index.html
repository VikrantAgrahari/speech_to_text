Convert Your Audio file into Text.

" This site is built using Python, Django, Speech Recognition Engine and ML to detect talking speech from Audio file and convert it to Text. "

You have recorded voice or your lecutrer notes and you want it to convert it to text instead of listening it all again, the work can easily be done here.

Two steps to go :

Press "Choose File" to upload audio file (make sure it is in .wav format)

And click Upload File button to start the identifying process

*Currently supports only   .wav   audio format only.*
